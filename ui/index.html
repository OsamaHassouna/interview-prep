    How it works
    
    
    Three independent meters track your progress.
    

    
    1 · Question difficulty (L1 – L5)
    
    
    Set per question. Drives XP per Q.
    
    
    
      
    L1
    Warm-up
    10 XP
    
      
    L2
    Standard
    25 XP
    
      
    L3
    Deep · senior depth
    50 XP
    
      
    L4
    Stretch · staff
    100 XP
    
      
    L5
    Boss-tier
    200 XP
    
    
    

    
    2 · Player level (top-bar XP meter)
    
    
    Accumulates from all XP earned. Streak bonus: +10% per Q at 3-in-a-row, +25% at 7-in-a-row.
    
    
    
      
    Apprentice
    0 XP
    
      
    Practitioner
    100 XP
    
      
    Senior
    300 XP
    
      
    Staff
    700 XP
    
      
    Principal
    1500 XP
    
      
    Distinguished
    3000 XP
    
    
    

    
    3 · Per-topic mastery (Bronze → Platinum)
    
    
    Shown as a tier badge on each topic card.
    
    
    
      
    Bronze
    5 correct at L2+
    
      
    Silver
    10 at L2+ incl. 3 at L3+
    
      
    Gold
    Silver + 5 at L4+
    
      
    Platinum
    Gold + Boss round 80%+
    
    
    

    
    Spaced repetition
    
    
      
      
    • Wrong → returns in 3 sessions
      • 
      
    • Partial → returns in 7 sessions
      • 
      
    • Correct → returns in 7, then 14, then 21 sessions
      • 
      
    • 3 correct in a row → retired (only re-surfaces in Boss rounds)
      • 
    
    

    
    Modes
    
    
    
      
    🎯 Daily Practice — due Qs from SR queue, then fresh ones
    
      
    📚 Topic Drill — pick a topic, optionally filter by track
    
      
    👹 Boss Round — 10 Qs in one topic, no skips, 80%+ earns Platinum eligibility
    
      
    🎙️ Mock Interview — 5 L3+ Qs from chosen track (UI / Angular)
    
      
    📈 Guided Path — leveled MCQ ladder. Pass a level (80%+) to unlock the next. Practical example at the end of each level.
